Arquitetura: conjunto de atributos visíveis ao computador
Organização de computadores: como isso é implementado logicamente

É importante estudar arquitetura e organização para saber lidar com problemas que possam surgir na raiz do zcódigo

Os principais componentes de um computador são: 
	cpu: tem a função de controlar a operação do computador e realizar o processamento de dados
	memória principal: armazenar dados e instruções
	I/O: permite ao usuário entrar e receber dados 

Programa armazenado:
	um computador que armazena dados do programa e dados de instruções
